8. TYPE CASTING

        // 8. TYPE CASTING

        /* Type Casting: is the process of converting
        a value from one data type to another. */

        // Implicit Type Casting (Automatically happens because there is no data loss)

        int numInt = 10;
        double numDouble = numInt; // 10.0

        System.out.println("number INT: " + numInt);
        System.out.println("Number DOUBLE: " + numDouble);

        // Explicit Type Casting (Manually convert values from one data type to another.
        // Used when converting from larger to smaller data types
        // Used when there is a possibility of data loss

        double pi3 = 3.14;
        System.out.println("PI as a double = " + pi3); // 3.14

        int intPI = (int) pi3;
        System.out.println("PI cast as an INT with data loss = " + intPI); // 3

        // Type Casting between different Data Types

        int myInt = 5;
        char charA = 'A';
        char typeCastChar = (char) (myInt + charA);
        int typeCastInt = myInt + charA; // Implicitly cast as an INT
        System.out.println(typeCastChar); // 'F'
        System.out.println(typeCastInt); // 70

Casting converts a value from one type to another. Widening conversions (int -> double) are automatic. Narrowing (double -> int) must be explicit and may lose data. Casting also helps show how chars map to numbers.

What’s happening here?

Key concepts

  • implicit cast (int -> double), explicit cast (double -> int), char/int arithmetic

When to use this

  • Use casting when you need to convert types, especially narrowing conversions.

Summary

  • Shows automatic widening, manual narrowing with data loss, and char/int conversions.

Common mistakes

  • Losing decimals when casting to int; unexpected results with char math; overflow.

Next step

9. OPERATORS

        // 9. OPERATORS

        // Arithmetic Operators + - * / %

        System.out.println(1 + 2); // 3
        System.out.println(2 * 2); // 4
        System.out.println(10/5); // 2
        System.out.println(10%5); // 0 (No Remainder. The left over after division)

        // Comparison Operators: == != > < >= <=

        int ten = 10;
        System.out.println(ten == 9); // false
        System.out.println(ten == 10); // true
        System.out.println(ten != 10); // false
        System.out.println(ten != 999); // true
        System.out.println(ten > 100); // false
        System.out.println(ten < 100); // true
        System.out.println(ten >= 999); // false
        System.out.println(ten >= 3); // true
        System.out.println(ten <= 3); // false
        System.out.println(ten <= 100); // true

        // Logical Operators: && || !

        boolean isSunny = true;
        boolean needUmbrella = false;

        boolean shouldGoOutside = isSunny && !needUmbrella; // true

        boolean canPlayGamesInside = isSunny || needUmbrella; // true

Arithmetic operators compute values (+ - * / %). Comparison operators produce booleans (== != > < >= <=). Logical operators combine booleans (&& || !) and are the backbone of decision-making.

15. BREAK AND CONTINUE KEYWORDS

Summary

19. ARRAYLIST

        // 19. ARRAYLIST

        // Create a Scanner object to read user input

        Scanner input = new Scanner(System.in);

        // Create an ArrayList to store the user input

        ArrayList<Integer> numbers = new ArrayList<>();

        // Initialize a variable to store the sum of the numbers

        int sum = 0;

        // Prompt the user to enter numbers and read them into the ArrayList

        System.out.println("Please enter numbers (type -101 to stop):");

        // Populate the ArrayList with user input

        while (true) {
            int num = input.nextInt();

            if (num == -101) {
                break;
            }
            numbers.add(num);
        }

        input.close();

        // Print the populated ArrayList of numbers

        System.out.println(numbers);

        // Check if the ArrayList is empty

        if (numbers.isEmpty()) {
            System.out.println("No numbers entered.");
            return;
        }

        // Add up the total of the numbers in the ArrayList

        for (int i = 0; i < numbers.size(); i++) {
            sum += numbers.get(i);
        }

        // Calculate the average of the numbers

        double average = (double) sum / numbers.size();

        // Print the average

        System.out.println("The average of the numbers is: " + average);

    } // endofmain

An ArrayList in Java is a resizable-array that provides a dynamic and flexible way to store and manage collections of objects. Unlike traditional arrays, which have a fixed size, an ArrayList can grow or shrink as elements are added or removed during runtime.

What’s happening here?

Key concepts

  • ArrayList<Integer>, Scanner, while(true), sentinel value (-101), add(), size(), get(), isEmpty(), for loop

When to use this

  • Use when you don’t know how many items you’ll collect and want a resizable list.

Summary

  • Read ints until -101, store them, handle empty input, sum values, compute and print average.

Common mistakes

  • Not checking isEmpty(); forgetting to cast for average; infinite loop if sentinel is never entered.

Next step
