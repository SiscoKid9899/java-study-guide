1. MAIN METHOD

    // 1. MAIN METHOD

main() is the entry point the JVM calls first. Anything inside this method runs in order. This is where your examples execute.

2. JAVA OUTPUT

        // 2. JAVA OUTPUT

        System.out.println("Hello my Friends!");

System.out.println(...) writes text to the console and then moves to a new line. This is the simplest way to verify your program is running and to inspect values.

3. VARIABLE INT

        // 3. VARIABLE INT

        int age = 44;
        System.out.println("My Age = " + age);

        age = 99;
        System.out.println("My age is now " + age);

An int stores whole numbers. You declare age, print it, reassign it, then print again. This demonstrates variable declaration, assignment, reassignment, and string concatenation.

4. WHOLE NUMBERS

        // 4. WHOLE NUMBERS

        byte x = 100;

        // short: -32,768 --> 32,767
        short y = 7777;

        // int: -214783648 --> 2147483647

Java has several whole-number types with different sizes/ranges (byte, short, int, long). Pick the smallest type that safely fits your values; long literals often use an L suffix.

5. DECIMAL NUMBERS

        // 5. DECIMAL NUMBERS

        // float: you should add "F" or "f" at the end of the number - 32 bits
        float pi = 3.14F;

        // Double: store decimal numbers with higher precision - 64 bits
        double pi2 = 3.14159;

float (32-bit) and double (64-bit) store decimals. Java treats decimal literals as double by default, so floats usually need an F suffix. doubles are typically preferred for precision.

7. CHAR

char holds a single UTF-16 code unit (one character in many common cases). Escape sequences like \n (newline) and \t (tab) represent special formatting characters.

8. TYPE CASTING

        // 8. TYPE CASTING

        /* Type Casting: is the process of converting
        a value from one data type to another. */

        // Implicit Type Casting (Automatically happens because there is no data loss)

        int numInt = 10;
        double numDouble = numInt; // 10.0

        System.out.println("number INT: " + numInt);
        System.out.println("Number DOUBLE: " + numDouble);

        // Explicit Type Casting (Manually convert values from one data type to another.
        // Used when converting from larger to smaller data types
        // Used when there is a possibility of data loss

        double pi3 = 3.14;
        System.out.println("PI as a double = " + pi3); // 3.14

        int intPI = (int) pi3;
        System.out.println("PI cast as an INT with data loss = " + intPI); // 3

        // Type Casting between different Data Types

        int myInt = 5;
        char charA = 'A';
        char typeCastChar = (char) (myInt + charA);
        int typeCastInt = myInt + charA; // Implicitly cast as an INT
        System.out.println(typeCastChar); // 'F'
        System.out.println(typeCastInt); // 70

Casting converts a value from one type to another. Widening conversions (int -> double) are automatic. Narrowing (double -> int) must be explicit and may lose data. Casting also helps show how chars map to numbers.

9. OPERATORS

        // 9. OPERATORS

        // Arithmetic Operators + - * / %

        System.out.println(1 + 2); // 3
        System.out.println(2 * 2); // 4
        System.out.println(10/5); // 2
        System.out.println(10%5); // 0 (No Remainder. The left over after division)

        // Comparison Operators: == != > < >= <=

        int ten = 10;
        System.out.println(ten == 9); // false
        System.out.println(ten == 10); // true
        System.out.println(ten != 10); // false
        System.out.println(ten != 999); // true
        System.out.println(ten > 100); // false
        System.out.println(ten < 100); // true
        System.out.println(ten >= 999); // false
        System.out.println(ten >= 3); // true
        System.out.println(ten <= 3); // false
        System.out.println(ten <= 100); // true

        // Logical Operators: && || !

        boolean isSunny = true;
        boolean needUmbrella = false;

        boolean shouldGoOutside = isSunny && !needUmbrella; // true

        boolean canPlayGamesInside = isSunny || needUmbrella; // true

Arithmetic operators compute values (+ - * / %). Comparison operators produce booleans (== != > < >= <=). Logical operators combine booleans (&& || !) and are the backbone of decision-making.

10. STRINGS

        // 10. STRINGS

        // Creating Strings

        String sayHello = "Hello ";
        System.out.println(sayHello); // Hello with a space at the end

        // Concatenating Strings: "+"

        String myFriends = "My Friends!";
        System.out.println(sayHello + myFriends); // Hello My Friends!

        // String Length

        int stringLength = myFriends.length();
        System.out.println("MyFriend string is " + stringLength + " characters long."); // 11

Strings are objects that store text. You can concatenate them with + and call methods like length(), toUpperCase(), and toLowerCase(). Use equals(...) for content comparison, not ==.

11. IF STATEMENT

        // 11. IF STATEMENT

        // If statement Else If statement and Else statement

        int myAge = 44;

        if (myAge >= 18) {
            System.out.println("You are an adult!");
        } else if (myAge >= 60) {
            System.out.println("Aged Person!");
        } else {
            System.out.println("You are a Minor");
        }

if/else-if/else evaluates conditions top-to-bottom and runs the first matching branch. Ordering matters: broader conditions should usually come after more specific ones.

12. SWITCH STATEMENT

        // 12. SWITCH STATEMENT

        int dayOfWeek = 4;

        switch (dayOfWeek) {

            case 0:
                System.out.println("Sunday");
                break;
            case 1:
                System.out.println("Monday");
                break;
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;
            case 4:
                System.out.println("Thursday");
                break;
            case 5:
                System.out.println("Friday");
                break;
            case 6:
                System.out.println("Saturday");
                break;
            default:
                System.out.println("Not A Day");
                break;
        }

switch selects one branch based on a single value. case labels define matches and break stops fall-through. default runs when no case matches.

13. WHILE LOOP

        // 13. WHILE LOOP

        int number = 0;

        while (number <= 5) {
            // Code to be executed repeatedly
            // as long as the condition is true
            System.out.println(number);
            number++;
        }

while repeats as long as its condition is true. Make sure something changes inside the loop so the condition eventually becomes false, preventing an infinite loop.

14. FOR LOOP

        // 14. FOR LOOP

        for (int i = 0; i <= 5; i++) {
            // initialization; condition; update
            // code to be executed repeatedly
            System.out.println(i);
        }

        System.out.println(sumUpNumbers()); // 55

for loops are ideal for counting and arrays. The structure is: initialize; condition; update. This runs predictably for a known number of iterations.

15. BREAK AND CONTINUE KEYWORDS

        // 15. BREAK AND CONTINUE KEYWORDS

        for (int i = 0; i < 10; i++) {

            if (i == 4) {
                break;
            }

            System.out.println(i);
        }

        for (int i = 0; i < 10; i++) {

            if (i == 4) {
                System.out.println("four");
                continue;
            }

            System.out.println(i);
        }

break exits the loop immediately. continue skips the rest of the current iteration and moves to the next one. Both are useful for controlling loop flow cleanly.

16. ARRAYS

        // 16. ARRAYS

        // dataType[] arrayName = new dataType[arraySize];

        int[] numbersArray = new int[5];


        // storing values in the array manually

        numbersArray[0] = 10;
        numbersArray[1] = 20;
        numbersArray[2] = 30;
        numbersArray[3] = 40;
        numbersArray[4] = 50;

        // storing values in the array with a for loop

        int[] numbersArray2 = new int[5];

        for (int i = 0; i < numbersArray2.length; i++) {
            numbersArray2[i] = i;
        }

        // Accessing elements in the array and printing them

        System.out.println("Element at index 0: " + numbersArray[0]); // Element at index 0: 10
        System.out.println(Arrays.toString(numbersArray2)); // [0, 1, 2, 3, 4]

Arrays store fixed-size sequences of the same type. Indexing starts at 0. You can populate arrays manually or with loops, and print them using Arrays.toString(...) for readability.

17. FOR EACH LOOP

        // 17. FOR EACH LOOP

        /* for(dataType element : collection/array) {
                code goes here
            } */

        int[] numbersArray3 = new int[5];

        for (int i = 0; i < numbersArray3.length; i++) {
            numbersArray3[i] = i;
        }

        for (double element : numbersArray3) {
            System.out.println(element); // prints out the array as doubles!
        }

The enhanced for loop iterates through each element without an index. It’s simpler for read-only iteration, but you don’t directly control the index unless you track it separately.

18. MULTI DIMENSIONAL ARRAYS

        // 18. MULTI DIMENSIONAL ARRAYS

        // Syntax:
        // dataType[][] arrayName = new dataType[rows][columns];

        // 3 rows and 3 columns

        int[][] myMatrix = new int[3][3];

        // Storing values in the 2D array

        // first row

        myMatrix[0][0] = 1;
        myMatrix[0][1] = 2;
        myMatrix[0][2] = 3;

        // second row

        myMatrix[1][0] = 4;
        myMatrix[1][1] = 5;
        myMatrix[1][2] = 6;

        // third row

        myMatrix[2][0] = 7;
        myMatrix[2][1] = 8;
        myMatrix[2][2] = 9;

        // Accessing elements in the 2D Array & printing them

        System.out.println(myMatrix[2][0]); // 7

        // Another way to populate the 2D array

        int[][] myMatrix2 = {
                {1,2,3},
                {4,5,6},
                {7,8,9}
        };

        // 2d array with nested for loops

        int[][] grid = new int[3][4]; // 3 rows, 4 columns

        for (int row = 0; row < grid.length; row++) {
            for (int col = 0; col < grid[row].length; col++) {
                grid[row][col] = row + col;
                System.out.print(grid[row][col] + " ");
            }
            System.out.println();
        }

        // Times Table 10 * 10

        int[][] table = new int[11][11];

        // populate the table and print it

        for (int i = 1; i < table.length; i++) {
            for (int j = 1; j < table[i].length; j++) {
                table[i][j] = i * j;
                System.out.printf("%4d", table[i][j]);
            }
            System.out.println();
        }

A 2D array is an array of arrays (rows and columns). Nested loops are the standard way to populate and print a grid. This pattern shows how row/col indexing works in practice.

19. ARRAYLIST

        // 19. ARRAYLIST

        // Create a Scanner object to read user input

        Scanner input = new Scanner(System.in);

        // Create an ArrayList to store the user input

        ArrayList<Integer> numbers = new ArrayList<>();

        // Initialize a variable to store the sum of the numbers

        int sum = 0;

        // Prompt the user to enter numbers and read them into the ArrayList

        System.out.println("Please enter numbers (type -101 to stop):");

        // Populate the ArrayList with user input

        while (true) {
            int num = input.nextInt();

            if (num == -101) {
                break;
            }
            numbers.add(num);
        }

        input.close();

        // Print the populated ArrayList of numbers

        System.out.println(numbers);

        // Check if the ArrayList is empty

        if (numbers.isEmpty()) {
            System.out.println("No numbers entered.");
            return;
        }

        // Add up the total of the numbers in the ArrayList

        for (int i = 0; i < numbers.size(); i++) {
            sum += numbers.get(i);
        }

        // Calculate the average of the numbers

        double average = (double) sum / numbers.size();

        // Print the average

        System.out.println("The average of the numbers is: " + average);

    } // endofmain

An ArrayList in Java is a resizable-array that provides a dynamic and flexible way to store and manage collections of objects.Unlike traditional arrays, which have a fixed size, an ArrayList can grow or shrink as elements are added or removed during runtime

20. METHODS

    // 20. METHODS

    public static int additionFunction() {

        int x = 7;
        int y = 3;

        return x + y; // 10
    }

    public int daysOfMonthSwitch() {

        String monthName = "May";
        int number_Of_DaysInMonth = 0;

        switch (monthName) {
            case "January":
            case "March":
            case "May":
            case "July":
            case "August":
            case "October":
            case "December":
                number_Of_DaysInMonth = 31;
                break;
            case "April":
            case "June":
            case "September":
            case "November":
                number_Of_DaysInMonth = 30;
                break;
            case "February":
                number_Of_DaysInMonth = 28;
                break;
            default:
                System.out.println("Not A Month");
                break;
        }
        return number_Of_DaysInMonth;
    }

    public static int sumUpNumbers(){

        int sum = 0;

        for (int i = 0; i <= 10; i++) {
            sum += i;
        }
        return sum;
    }
}

Methods (functions) let you reuse logic. A return type like int means the method produces a value. Parameters (if present) would let you pass input in; return sends output back to the caller.
